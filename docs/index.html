Vecino Seguro

Vecino Seguro lets residents report urban problems quickly. The app supports Sustainable Development Goal 11: “Sustainable Cities and Communities.” It stores report data in Firebase and sends a WhatsApp message to local authorities. Residents in periurban areas (for example, Tlajomulco de Zúñiga, Jalisco) can report potholes, broken streetlights, water leaks, and other issues.

2. Install and run the app

  1. Clone the repo and open the project folder: 
    git clone https://github.com/<YOUR_USERNAME>/vecino_seguro.git
cd vecino_seguro
  2. Install Flutter (3.29.2 or later) and Dart (3.7.2). To verify, run: 
    flutter --version
    If you need to switch channels or upgrade, run: 
    flutter channel stable
flutter upgrade
flutter version 3.29.2
  3. Install dependencies: 
    flutter pub get
  4. Add Firebase credentials:
    • Android: place google-services.json in android/app/.
    • iOS: place GoogleService-Info.plist in ios/Runner/.
  5. Run the app on an emulator or device: 
    flutter run
    (In Android Studio, open AVD Manager, select an emulator, and click Run.)
  6. Build a release APK: 
    flutter build apk --release
    Find the APK at build/app/outputs/flutter-apk/app-release.apk.

3. Main features

  • Anonymous authentication
    The app creates an anonymous user in Firebase Auth. You don’t need to sign up. Firebase Auth manages read/write permissions in Firestore and Storage.
  • Create a report
    • Select an incident type: “Pothole,” “Streetlight,” “Trash,” “Water leak,” or “Other.”
    • Enter the location and description.
    • Optionally take a photo with the camera.
    • Tap “Send report” to save data in Firestore. Optionally, open WhatsApp to share: 
      https://wa.me/5214626214305?text=<encoded_text>
  • View reports
    • “My reports” screen shows all reports sorted by date.
    • Filter by status: “All,” “Pending,” or “Resolved.”
    • An anonymous user can delete their own reports (if enabled).
  • About CEFODEH
    Displays information about CEFODEH. Tap buttons to contact via WhatsApp or email.

4. App screens

4.1. Splash screen

When you launch the app, it displays the CEFODEH logo and the text “Centro de Formación para el Desarrollo Humano” for 3 seconds.

4.2. Home

Main screen with buttons for “Create report,” “View reports,” and “About CEFODEH.”

4.3. Create report

Form to select incident type, enter location and description, and optionally take a photo.

4.4. View reports

List of reports with a status filter and a delete option for each report.

4.5. About CEFODEH

Institutional information with buttons to contact via WhatsApp and email.
