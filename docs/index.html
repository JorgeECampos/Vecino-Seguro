Vecino Seguro

Vecino Seguro helps residents report urban incidents quickly. It supports Sustainable Development Goal 11: “Sustainable Cities and Communities.” Using Firebase, the app stores incident data and notifies authorities via WhatsApp. Residents in periurban areas (for example, Tlajomulco de Zúñiga, Jalisco) can report issues such as potholes, broken streetlights, or water leaks.

2. Installation and use

  1. Clone the repository and navigate to the project folder: 
    git clone https://github.com/<YOUR_USERNAME>/vecino_seguro.git
cd vecino_seguro
  2. Make sure you have Flutter 3.29.2 (or higher) and Dart 3.7.2 installed. You can check with: 
    flutter --version
    If you need to switch channels or upgrade, for example: 
    flutter channel stable
flutter upgrade
flutter version 3.29.2
  3. Install Dart/Flutter dependencies: 
    flutter pub get
  4. Make sure you have configured the file android/app/google-services.json (for Android) and ios/Runner/GoogleService-Info.plist (for iOS) with your Firebase project credentials.
  5. Run the application on an emulator or a real device: 
    flutter run
    (In Android Studio, open the AVD Manager, select an emulator, and press Run.)
  6. To build a production-ready APK: 
    flutter build apk --release
    The generated APK will be located at build/app/outputs/flutter-apk/app-release.apk.

3. Main features

  • Anonymous authentication
    Each user accesses the app without registering. Firebase Auth creates an anonymous user to manage read/write permissions in Firestore and Storage.
  • Report creation
    • Select an incident type: “Pothole,” “Streetlight,” “Trash,” “Water leak,” or “Other.”
    • Enter the location and description.
    • Optionally take a photo with the camera.
    • When you tap “Send report,” the data is saved to Firestore and, optionally, WhatsApp opens to share the incident: 
      https://wa.me/5214626214305?text=<encoded_text>
  • Report list
    • “My reports” screen: shows all reports sorted by date.
    • Status filter: “All,” “Pending,” or “Resolved.”
    • An anonymous user can delete their own reports (if enabled).
  • About CEFODEH
    Displays information about the organization (CEFODEH), along with buttons to send messages via WhatsApp and email.

Main screen with buttons for “Create report,” “View reports,” and “About CEFODEH.”

List of reports with a status filter and an option to delete each one.

5. How to contribute

If you want to help improve Vecino Seguro, follow these steps:

  1. Fork this repository.
  2. Create a branch for your new feature or fix: 
    git checkout -b my-new-feature
  3. Make your changes, then: 
    git add .
git commit -m "Brief description of your improvement"
  4. Push your branch to the remote repository: 
    git push origin my-new-feature
  5. Open a Pull Request on GitHub from your branch to main in this repository.
