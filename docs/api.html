📡 API de Vecino Seguro

Consulta pública de reportes ciudadanos

🔹 Endpoint disponible

GET https://us-central1-mi-app-vecinal.cloudfunctions.net/getReportes

📘 Descripción

Devuelve un listado de reportes públicos creados por los usuarios de la app Vecino Seguro.

🔐 Autenticación

No requiere autenticación. Acceso público.

📥 Parámetros

Este endpoint no acepta parámetros de entrada.

📤 Respuesta

Un arreglo JSON de objetos con esta estructura:

{
  "tipo": "Bache",
  "ubicacion": "Av. Hidalgo #123",
  "descripcion": "Hay un bache profundo",
  "estado": "pendiente",
  "fecha": "2025-06-01T14:30:00.000Z",
  "fotoURL": "https://firebasestorage.googleapis.com/..."
}

📦 Código de estado esperado

  • 200 OK – Respuesta exitosa
  • 500 Internal Server Error – Error en la función
