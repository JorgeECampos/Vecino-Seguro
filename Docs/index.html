# Vecino Seguro **“Vecino Seguro”** es una app Flutter que permite a cualquier ciudadano reportar problemas en su comunidad (baches, luminarias fundidas, basura, fugas, etc.) y enviarlos automáticamente por WhatsApp y almacenarlos en Firestore / Firebase Storage. --- ## Tabla de contenidos 1. [Pantallas y ejemplos](#pantallas-y-ejemplos) 5. [Cómo contribuir](#cómo-contribuir) 6. [Licencia](#licencia) --- ## Descripción del proyecto La aplicación **Vecino Seguro** nació para abordar el ODS 11: “Ciudades y comunidades sostenibles”. Permite que los residentes de una zona periurbana (por ejemplo, Tlajomulco de Zúñiga, Jalisco) reporten incidencias urbanas de forma rápida, almacenando la información en Firebase y notificando a las autoridades vía WhatsApp. --- ## Instalación y uso 1. Clona el repositorio en tu máquina: ```bash git clone https://github.com//vecino_seguro.git cd vecino_seguro ## Características principales Autenticación anónima: Cada usuario accede sin necesidad de registrarse; Firebase Auth crea un usuario anónimo que permite controlar permisos de lectura/escritura en Firestore y Storage. Creación de reportes: - El usuario selecciona el tipo de incidencia (bache, luminaria, basura, fuga de agua, otro). - Llena ubicación y descripción. - Puede tomar una foto opcional con la cámara. - Al presionar “Enviar reporte”, se guarda en Firestore y, opcionalmente, se envía por WhatsApp. Visualización de reportes: - Pantalla “Ver reportes” muestra todos los reportes ordenados por fecha. - Filtrado por estado (pendiente / resuelto). - El usuario anónimo puede borrar sólo sus propios documentos si implementas validación en Firestore (ver secciones avanzadas). Sección “Acerca de CEFODEH”: - Muestra información de la organización responsable (CEFODEH). - Botones para enviar mensaje por WhatsApp o correo. ## Pantallas y ejemplos ### Splash Screen 1. Home 2.
